Coffee Run was the result of a project in a course I took at KTH called Mobile Develpment With Web Technologies. We built a web app that allows people to place orders for coffee on a map, only to have others pick it up for them in exchange for a sum of money decided by the person ordering. The app was built using regular HTML, CSS and JavaScript and uses APIs from Firebase and Google Maps for user data and map functionality. We also used OnsenUI for some elements in the app.

My contributions to this project was mainly to implement support for Firebase, in order for the interactivity to work as intended.
